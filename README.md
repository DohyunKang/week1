# NI card를 이용한 Digital Input / Output & Analog Input 제어

## 설명
NI card와 measurement&WPF를 활용하여 DO/DI 제어 및 AI Graph 구현

## 기능
- Switch on/off 로 DO 제어
- DO를 통해 DI에 들어온 신호로 LED 제어
- AI의 값을 Graph로 표현
